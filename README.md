## 프로젝트 구성
이전 백오피스 프로젝트 기반으로 구현한 레포지토리입니다.

오늘의 집 ui 클론 코딩 중입니다.

학습을 위해 꾸준히 업데이트 예정입니다.

- Next 14
- TypeORM -> Prisma
- MariaDB -> MongoDB
- Graphql
- Vercel 배포

## 실행하기

```bash
npm run dev
# or
yarn dev
```
